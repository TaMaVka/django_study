# VisualIQ — Тренажер визуальной насмотренности

Образовательное веб-приложение для тренировки навыков визуального восприятия цвета, чувства пропорций и изучения теории дизайна. 

Проект разработан в рамках учебного курса по разработке WEB-приложений на Django.

##  Возможности проекта

| Функция | Описание |
|---------|----------|
| **HEX-Снайпер** | Тренажер: угадайте HEX-код случайно сгенерированного цвета на глаз. |
| **HSL-Мастер** | Тренажер: подберите параметры H/S/L с помощью ползунков, чтобы воссоздать цвет. |
| **База знаний** | Образовательные статьи (Markdown) по теории цвета и типографике. |
| **Геймификация** | Экономика: зарабатывайте монеты в мини-играх, чтобы покупать доступ к закрытым статьям. |
| **Пользовательский контент** | Возможность предложить свою шпаргалку через форму (CRUD). |

##  Стек технологий

- **Backend:** Python 3.10+ / Django 5.x 
- **База данных:** SQLite (встроенная)
- **Frontend:** HTML5 / Bootstrap 5 (CDN) / django-crispy-forms
- **Дополнительно:** Pillow (для работы с медиа), Markdown

##  Инструкция по локальному запуску

Для запуска проекта на вашем компьютере выполните следующие шаги в терминале.

### 1. Клонирование и настройка окружения
```bash
# Клонируйте репозиторий
git clone https://github.com/TaMaVka/django_study.git
cd django_study

# Создайте виртуальное окружение
python3 -m venv venv

# Активируйте виртуальное окружение
# Для Linux/macOS:
source venv/bin/activate
# Для Windows:
# venv\Scripts\activate

# Установите зависимости
pip install -r requirements.txt
```

### 2. Настройка базы данных
Перед первым запуском нужно создать таблицы и суперпользователя:
```bash
# Применение миграций базы данных
python manage.py makemigrations
python manage.py migrate

# Создание панели администратора (потребуется ввести логин, email и пароль)
python manage.py createsuperuser
```

### 3. Запуск сервера
```bash
# Запуск локального сервера разработки
python manage.py runserver
```

После этого откройте браузер и перейдите по адресу: **[http://127.0.0.1:8000](http://127.0.0.1:8000)**.
Панель администратора (для добавления первых статей) доступна по адресу: **[http://127.0.0.1:8000/admin/](http://127.0.0.1:8000/admin/)**.
